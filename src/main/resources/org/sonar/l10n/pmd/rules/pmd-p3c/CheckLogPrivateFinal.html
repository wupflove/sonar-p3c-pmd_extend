LOGGER对象申明修饰符必须是 private static final

案例：
浙江CRM系统中心化项目组，在迁移代码的过程中很多Log的误用，导致后来在定位问题的时候比较困难。代码规范
说明
没有使用final定义Log变量，后续log对象被修改，导致打印出的日志信息有误
影响范围
导致在跟踪日志的时候找不到日志或者找到的日志有问题，有产生误导。如果不适用final关键字可能会导致线程不安全，影响对象序列化

错误代码实例
public class MsgProducerSVImpl implements IMsgProducerSV {
private static transient Log log = LogFactory.getLog(Order2BillingSVImpl.class);
正确代码实例
public class MsgProducerSVImpl implements IMsgProducerSV {
private final static transient Log log = LogFactory.getLog(Order2BillingSVImpl.class);
