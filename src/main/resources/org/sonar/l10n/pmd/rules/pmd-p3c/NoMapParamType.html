禁止使用MAP类型作为参数

案例：
大量开发人员在开发服务接口时，使用Map作为出参和入参，导致接口自测、运维、后续开发非常困难

错误写法
private static final Logger logger = LoggerFactory.getLogger(EnquiryApproveController.class);
@ResponseBody
@RequestMapping(value = "/submitSales",method = RequestMethod.POST)
@ApiOperation(value = "审批提交", httpMethod = "POST")
public Response submitSales(@RequestBody Map reqMap, Principal principal)

正确写法
@RequestMapping("findSaleTaskDistArea")
public BaseResponse> findSaleTaskDistArea(@RequestBody SaleTaskDistAreaRequst request)
